1% log pdf of normal(정규분포의 확률밀도함수의 로그값)
 2% x = normal variates
 3% mu = vector of means
 4% sig2vec = vector of variances
 5% (관측치간 자기상관이 없다고 가정하므로 개별 관측치마다 우도함수값을 계산하는 것)
 6function [retf] = lnpdfn(x, mu, sig2vec)
 7
 8sig2vec = real(sig2vec);
 9if minc(sig2vec) < 0
10    error('sigma should be positive');
11end
12c = -0.5*log(2*sig2vec*pi);
13e = x - mu;
14e2 = e.*e;
15
16retf = c - 0.5*e2./sig2vec;
17end
